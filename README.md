# How to run

## Backend : 

> Server File

- install all the packages are needed, so in terminal: 

```
$ npm i
$ nodemon

````

- the server will be running on port 3000 


## Frontend :

> Client File

- install all the packages are needed, so in terminal: 

```
$ npm i
$ npm start

````
- the frontend must be running on another port
